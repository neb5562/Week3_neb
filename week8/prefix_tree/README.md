# Prefix tree!
Use  **demo.rb** to test Prefix Tree.
To install as a **Gem**:

 - `gem 'ptree', '~> 0.0.2'`
 - `gem install ptree`

Use  **ptree** gem.
- `require 'node'`
- `var = Node.new`
- `var.add('word')`
- `var.delete('word')`
- `var.list`
- `var.list('wo')`
- `var.include?('wor')`
- `var.find('word')`
[Link to official gem website](https://rubygems.org/gems/ptree)
